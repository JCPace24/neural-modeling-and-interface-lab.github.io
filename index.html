On Memory Neurotechnologies

We develop brain-like devices that can mimic and restore cognitive functions.

To pursue this goal, we use a combined experimental, theoretical, and computational strategy to develop next-generation modeling and neural interface methodologies; investigate brain functions during natural behaviors; understand how neural systems, such as the hippocampus, perform higher-order cognitive functions; and build hippocampal memory prostheses that can restore memory functions lost in diseases or injuries.
